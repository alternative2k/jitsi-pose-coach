# Jitsi Pose Coach

Real-time fitness coaching with pose detection over video conference.

## Quick Start

```bash
# Deploy to VPS
chmod +x deploy.sh
./deploy.sh
```

## Files

- `deploy.sh` - One-click VPS setup
- `nginx.conf` - Nginx proxy configuration
- `pose-room.html` - Web interface
- `docker-compose.override.yml` - Jitsi customization
- `.env.example` - Environment template

## Access

After deployment, access at VPS IP address over HTTP.
